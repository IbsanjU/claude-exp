# Tutorial Platform

Interactive tutorial website focused on JavaScript first, designed to expand into Python, Java, C++, Go, and Playwright.

## Goals

- Teach with hands-on, runnable code in lessons.
- Keep code execution secure by default.
- Deploy core web app on Vercel.
- Support optional login for progress tracking.
- Enable future personalized learning paths.

## Scope (Current)

This folder currently contains product and engineering planning artifacts:

- `docs/IMPLEMENTATION_PLAN.md`
- `docs/PROGRESS_TRACKER.md`
- `docs/ARCHITECTURE.md`
